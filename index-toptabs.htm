• What is tabby-css?

    tabby-css is a fork of Jason Shi's TrueTabs CSS3 tabbed content switcher. Jason's CodePen is excellent, but I wanted to make the CSS more modular. So I separated the basic functionality, and moved design flourishes into separate CSS files. I also removed the need to change classes to change design; design changes happen in the CSS. The basic functionality is provided by tabby.css, base layout design, fonts and main animations are provided by tabby-{LAYOUT}-.css, and color skinning and flourishes ('makeup') are in a separate CSS file named as tabby--{MAKEUP}.css.   • Configuration

    The CSS is split into several files, for a more modular configuration. It is assumed the deployed website will do one of these:

    • Concate all css into one file before deployment, either by and or via build tool like Gulp
    • Concate all css after deployment, using jsDelivr CDN or a server-side solution.
    • Link to each individual CSS file (not recomended outside of development)

    It has many color schemes, as well as animations on tab changes. It also has typography settings, a grid system, and, best of all, is completely responsive. It also has nice browser support, with support for IE9+, Chrome, Firefox, Opera, and Safari. The code is pretty commented and you should be able to find and edit stuff with no problems. I am planning on improving on this even further, so it will only get better from here
